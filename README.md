🇨🇦 Historical Gas Price Trends in Canada — Power BI Dashboard

Overview
This project analyzes over 10 years of public fuel price data across 12 Canadian provinces to uncover regional price volatility, inflation trends, and inter-provincial disparities. Built using Power BI with advanced DAX measures and Power Query transformations.

Tools Used
Power BI | Power Query | DAX | Excel

Key Findings

Identified price variance of up to 18% across Canadian provinces
Built 4 KPI dashboards to compare regional volatility and long-term inflation trends
Cleaned and transformed 10+ years of raw public fuel data using Power Query

Features

Regional price comparison across all 12 provinces
Year-over-year inflation trend analysis
Interactive slicers by province, and year

How to Open
Download the .pbix file from this repository
Open with Power BI Desktop (free)
Use the slicers to explore regional and time-based trends

Data Source
Public fuel price data — Government of Canada / Natural Resources Canada
